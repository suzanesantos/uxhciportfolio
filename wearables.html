Topic Assignment - Concept


This wearable concept is based on glasses that would provide some mobile features like a speaker, microphone, and MicroLED displays. That would allow the user to see the person’s picture of who they are talking to. The glasses supporters would have ultra-thin Piezo film speakers so they would hear also music. A microphone module would be placed in the nose’s glass supporter. Videos could be also displayed on one small frame displayed on the glasses. A Wireless Module Transceiver would be used to connect to the mobile.



  • The interaction

    • The glasses would work on voice command, motion gesture and touch. The user could receive the call by saying “accept call”. This type of command would be also applied to access the internet. Pages could be scrolled could be through the touchpad.





  • Sensors

    • The main sensors are the microphone, touch sensor placed on the touchpad and motion sensor connected camera. During a the microphone can captures the sound emitted by the user.


  • Actuators

    • The main actuators are touch from the user’s finger, sound from the voice commands and movement of the heads motion.



  • Features

    • The glasses would be available in 5 trendy colors and would be possible to charge it in wireless.

Arduino - Part 1


For 2 weeks I worked with Arduino, which is an open-source electronics platform for building prototypes. It gave us, students, the opportunity to practice through some exercises which are displayed throughout this page. Arduino combines hardware with software and is suitable to assist in designing interactive environments and objects.



    Exercise 3 B

    This exercised required that a LED lamps alternate the power. That was possible by coding a loop function.

    For the first second, the green LED lamp, connected to pin 13, will have the voltage level HIGH and after 1 second LOW. This will be reversed for the other LED lamp. For the yellow LED lamp, connected to pin 12, the voltage will be LOW and after one second it will be HIGH. This loop creates then an alternate sequence.


    Arduino Code:

    // the setup function runs once when you press reset or power the board
void setup() {
 pinMode(13, OUTPUT);
 pinMode(12, OUTPUT);
}

//the loop function runs over and over again forever
void loop() {
 digitalWrite(13,HIGH); // turn the LED on
 digitalWrite(13,LOW);  // turn the LED off
 delay(1000);   // wait for a second

 digitalWrite(13,LOW);
 digitalWrite(13,HIGH);
 delay(1000);   // wait for a second
 }
}



    Exercise 4 C

    The task here required that one LED slowly fades in and the other fades out simultaneously. In the code, 255 is the maximum brightness power it can get. The brightness of the green LED pin increases gradually until it reaches its maximum. For the yellow LED pin, it will work reversely. It will happen by setting it to start from the maximum brightness, 255, minus the brightness integer. As result, one LED pin fades out as its voltage decreases gradually while the other LED pin fades in as its voltage increases.


    Arduino Code:

    
 int greenLedPin = 10;
 int yellowLedPin = 9;

 void setup() {
  pinMode(greenLedPin, OUTPUT);
  pinMode(yellowLedPin, OUTPUT);
 }

 void loop() {
  for (int brightness=0; brightness <256; brightness++){
  analogWrite(greenLedPin, brightness);
  analogWrite(yellowLedPin, 255-brightness); //
  delay(10);
  }
 }



    Exercise 6 C

    Here the task required that the green LED pin and the yellow LED pin power alternate, as they fade in and out. That was possible by using the input of a potentiometer, to power the LED pins and print out the sensor values. This exercise showed how to employ an analog input to regulate an analog output. In order to alternate the output, the brightness equation of Exercise 4 was added to create this effect.


    Arduino Code:

    float sensorValue = 0;
int sensorPin = A0;
int greenLed = 10;
int yellowLed = 9;


void setup() {
pinMode(sensorPin, INPUT);  // set the the sensor pin to input
pinMode(greenLed, OUTPUT);  // set the LED pin to output
pinMode(yellowLed, OUTPUT); // set the LED pin to output
Serial.begin(9600);         // start a serial connection
}

void loop() {
 sensorValue = analogRead(sensorPin); // read the value/current on the sensor pin and
                                      // store that value in the variable sensorValue
 sensorValue = map(sensorValue, 0, 1023, 0, 255); // rescales the sensor value to 0-255
 Serial.print("sensor = "); // print out the text "sensor = " to the serial
                  
 Serial.println(sensorValue); // print out sensorValue to the Serial Monitor
 
analogWrite(greenLed, sensorValue);  // send power to the LED
analogWrite(yellowLed, sensorValue); // send power to the LED
 
 delay(20); // delay for 0.2 seconds
 
for (int brightness=0; brightness <256; brightness++){
 analogWrite(greenLed, brightness);
 analogWrite(yellowLed, 255-brightness); //
 delay(10);
 }
}



    Exercise 7 A

    This exercise was simple but funny. Here we start to see more than lights blinking, we see images being displayed. That was achievable with a LED Matrix component. To configure it we used a couple of functions, such as setRow (for the rows) and setCol (for the columns). A code was provided so we could make a smiley face and a frowny face be displayed on the LED matrix. The image displayed will alternate each second.


    Arduino Code:

    #include "LedControl.h"
/*
DIN connects to pin 12
CLK connects to pin 11
CS connects to pin 10
*/
LedControl lc = LedControl(12,11,10,1); // create a new LedControl object
void setup() {
 lc.shutdown(0,false); // Turn matrix on, no power saving
 lc.setIntensity(0,8); // Set brightness to a medium value
 lc.clearDisplay(0); // Clear the display
}
void loop(){
lc.setRow(0,0,B00111100); // set the first row of the matrix
 lc.setRow(0,1,B01000010); // set the second row of the matrix
 lc.setRow(0,2,B10100101); // etc.
 lc.setRow(0,3,B10000001);
 lc.setRow(0,4,B10100101);
 lc.setRow(0,5,B10011001);
 lc.setRow(0,6,B01000010);
 lc.setRow(0,7,B00111100);
 delay(1000);
 lc.setRow(0,0,B00111100);
 lc.setRow(0,1,B01000010);
 lc.setRow(0,2,B10100101);
 lc.setRow(0,3,B10000001);
 lc.setRow(0,4,B10011001);

}
